## Development guide

### Setting up a development environment

**Please prepare:**

- [Node.js](https://nodejs.org/en)
- [pnpm](http://pnpm.io/)

We will use [SolidStart](https://start.solidjs.com/) and [TanStack Router](https://tanstack.com/router/latest/docs/framework/solid/overview) with [TypeScript](https://www.typescriptlang.org/) for this project. The package manager used for contributions **must be pnpm**.

### Scripts

- **`dev`**: to start development server.
- **`build`**: to build this project locally.
- **`start`**: to start production server locally.
- **`version`**: see vinxi version.
- **`format:check`**: format check with Prettier.
- **`format:write`**: format code with Prettier (write command).
- **`lint`**: Lint code with ESLint.
- **`prepare`**: Prepare script for Husky.
- **`lint-staged`**: Script for lint-staged.
